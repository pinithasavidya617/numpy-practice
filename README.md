#  NumPy For Data Analysis 

A hands-on collection of NumPy scripts covering core concepts used in data analysis and numerical computing.

---

##  What This Repo Covers

* Array creation and properties
* Indexing, slicing, and memory behavior
* Broadcasting and vectorized operations
* Mathematical and statistical functions
* Linear algebra basics
* Sorting, searching, and random data generation
* Performance optimization using NumPy

---

##  Structure

Each script focuses on a specific concept and can be run independently.

---

##  Key Focus

* Writing efficient, vectorized code
* Understanding how NumPy works under the hood
* Building a strong foundation for data analysis and machine learning

---

##  Tech Stack

* Python
* NumPy
